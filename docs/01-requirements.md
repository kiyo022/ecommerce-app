# E-コマース　アプリケーション　要件定義書

## プロジェクト概要

React × TypeScript × Supabase × Stripeを用いた小~中規模Eコマースアプリ

顧客情報の登録、商品検索・購入・注文管理を一元管理を行えるプラットフォーム

## 主な機能

### ユーザー機能

- [ ] ユーザー登録・ログイン（Supabase Auth）
- [ ] ユーザー情報管理
- [ ] 注文履歴閲覧

### 商品機能（デフォルト）

- [ ] 商品一覧表示(ページネーション)
- [ ] 商品検索・フィルタリング
- [ ] 商品詳細表示
- [ ] 商品画像表示(複数の場合、カルーセル)
- [ ] 商品レビュー表示

### カート・注文機能

- [ ] ショッピングカート（フロント管理）
- [ ] 注文確定
- [ ] 注文履歴管理

### 決済機能

- [ ] Stripe統合
- [ ] 決済処理
- [ ] 決済確認メール送信

### 管理機能

- [ ] 商品管理(追加・編集・削除)
- [ ] 画像D&Dアップデート
- [ ] 注文管理
