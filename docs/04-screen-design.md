# E-コマース 画面設計書

## 📌 概要

E-コマースアプリの全画面設計及び画面遷移図

---

## 🗺️ 画面一覧

### ユーザー向け画面

| 画面名               | URL            | 説明                   |
| -------------------- | -------------- | ---------------------- |
| トップページ         | /              | 商品一覧表示・検索     |
| 商品詳細ページ       | /products/{id} | 商品詳細・レビュー表示 |
| カートページ         | /cart          | ショッピングカート     |
| チェックアウト       | /checkout      | 注文確認・決済         |
| 注文確認ページ       | /orders/{id}   | 注文詳細確認           |
| 注文履歴ページ       | /orders        | 過去の注文一覧         |
| ユーザープロフィール | /profile       | プロフィール編集       |
| 認証ページ           | /auth/login    | ログイン               |
| 登録ページ           | /auth/signup   | 新規登録               |

### 管理者向け画面

| 画面名         | URL                       | 説明                     |
| -------------- | ------------------------- | ------------------------ |
| 管理画面トップ | /admin                    | ダッシュボード           |
| 商品管理       | /admin/products           | 商品一覧・CRUD           |
| 商品編集       | /admin/products/{id}/edit | 商品情報編集             |
| 注文管理       | /admin/orders             | 注文一覧・ステータス管理 |

---

## 📊 画面遷移図

```
┌─────────────────────────────────────────────────────────────┐
│                      トップページ                            │
│                    (商品一覧・検索)                          │
└────────┬──────────────────────────────────┬─────────────────┘
         │                                  │
         ├─→ 商品詳細ページ ────→ レビュー投稿
         │         │
         │         └─→ カート追加
         │
    ログイン/登録
         │
         ├─→ ユーザープロフィール
         │         │
         │         └─→ 注文履歴
         │              │
         │              └─→ 注文詳細
         │
         └─→ カート ────→ チェックアウト ────→ 注文確認

【管理者】
ログイン → 管理画面トップ
           ├─→ 商品管理 ────→ 商品編集
           └─→ 注文管理 ────→ ステータス更新
```

### 1️⃣ トップページ（/）

**目的:** 商品一覧表示・検索

**UI コンポーネント:**

```
┌─────────────────────────────────────┐
│       Navbar                         │
│  [Logo] [検索] [カート] [ログイン]   │
├─────────────────────────────────────┤
│                                     │
│  【フィルタ・ソート】                │
│  [カテゴリ] [価格順] [新着順]       │
│                                     │
├─────────────────────────────────────┤
│                                     │
│  商品カード  商品カード  商品カード   │
│  [画像]     [画像]     [画像]       │
│  [名前]     [名前]     [名前]       │
│  [¥価格]    [¥価格]    [¥価格]      │
│  [★★★★★] [★★★★☆] [★★★☆☆]     │
│                                     │
├─────────────────────────────────────┤
│  [< 前へ] [1] [2] [3] [次へ >]      │
│           ページネーション           │
└─────────────────────────────────────┘
```

**機能:**

- 商品一覧を表示（ページネーション）
- カテゴリでフィルタリング

- ソート（新着順・価格順など）

- GET /products（クエリパラメータ付き）
